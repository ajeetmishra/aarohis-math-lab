### **The "Down Arrow" Philosophy**
Unlike standard quizzes that just tell a child if they are "Right" or "Wrong," this lab uses a **Three-Step Reveal** to build mental scaffolding:
1.  **The Challenge:** A problem is presented (e.g., $99 \times 11$).
2.  **The "Think" Bridge:** A logic hint appears to guide the mental process (e.g., "Think: $100 \times 10$").
3.  **The Result:** The actual answer is revealed alongside the estimate to build confidence.

### **Architecture**
This project is a **"Static Web Suite."** It requires no database, no backend, and no complicated setup.
* **Stack:** Pure HTML5, CSS3 (with Responsive Design), and Vanilla JavaScript.
* **Responsiveness:** Includes a "Mobile-First" override that hides settings behind a hamburger menu on smaller screens, maximizing focus on the problem.
* **Hosting:** Optimized for **GitHub Pages**. Simply upload the files to a repository and enable "Pages" in the settings to get your live URL.

### **Installation**
1.  Clone this repository: `git clone https://github.com/yourusername/aarohis-math-lab.git`
2.  Open `index.html` in any modern web browser.
3.  To host it yourself, simply drag and drop the files into a GitHub repository and turn on GitHub Pages.
